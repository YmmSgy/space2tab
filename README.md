# space2tab
Converts start-of-line spaces to tabs

## Usage
`space2tab [OPTIONS] FILE...`

### `OPTIONS`
Option      | Description
------------|----------------------------------
`-h`        | Display this help message
`-V`        | Display version info
`-v`        | Enable verbose mode
`-s N`      | Change tabsize to N, default is 8

### `FILE...`
List of files to convert
